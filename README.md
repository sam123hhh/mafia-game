# ليلة المافيا — موقع إدارة الليل

## الملفات
- `index.html` — هيكل الصفحة (يحمّل style.css و app.js)
- `style.css` — كل التصميم (ألوان، خطوط، أنيميشن)
- `app.js` — كل منطق اللعبة، مربوط بقاعدة بيانات Firebase Realtime Database

## كيف تجربه محليًا (على جهازك)
1. افتح المجلد بـ VS Code.
2. نزّل إضافة **Live Server** من الـ Extensions (لو ما عندك).
3. دوس كليك يمين على `index.html` → **Open with Live Server**.
4. رح يفتحلك المتصفح على رابط شبيه بـ `http://127.0.0.1:5500`.

⚠️ ما تفتح `index.html` بدبل-كليك مباشرة (file://) — لازم يكون عن طريق سيرفر محلي (متل Live Server) عشان الاستيراد `import` يشتغل صح.

## كيف تنشره عشان تستخدمه من جوالاتكم بالخارج (مجانًا)
أسهل طريقة: **GitHub Pages**
1. اعمل حساب على github.com (لو ما عندك).
2. اعمل Repository جديد (مثلاً `mafia-game`).
3. ارفع الثلاث ملفات (`index.html`, `style.css`, `app.js`) للـ Repository.
4. من إعدادات الـ Repository → **Pages** → اختار الفرع `main` والمجلد `/ (root)` → Save.
5. بعد دقيقة أو اثنين رح يعطيك رابط شبيه بـ:
   `https://your-username.github.io/mafia-game/`
6. هاد الرابط ثابت، افتحوه من أي جوال بدون أي حساب.

## ملاحظة أمان بسيطة
قاعدة البيانات حاليًا شغالة بـ "Test Mode" (مفتوحة). هاد تمام لعدد صغير متل مجموعتكم (أقل من ٢٠)، بس بعد ٣٠ يوم Firebase بتقفل الوضع هاد تلقائيًا. إذا صار وقتها، رجع اسألني ومنظبط "Security Rules" بسيطة تخلي الموقع يشتغل عادي بدون فتح قاعدة البيانات للعالم.
